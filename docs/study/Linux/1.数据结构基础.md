## 1.1 基础概念和术语
### 1.1.1 基础概念
**数据**：能输入到计算机中描述客观事物的符号，包括文本、声音、图像、符号等。

+ 例如，全班学生的信息表、传感器采集的原始读数。

**数据元素**：数据的基本单位，也称节点或记录。

+ 例如，学生信息表中的一行记录（如“张三，20，男”）。

**数据项**：数据元素的最小不可分割元素，也称域。

+ 例如，学生信息表中学生的“学号”。

**数据对象**：性质相同的数据元素的集合，是数据的子集。

+ 例如，所有整数构成的集合。
